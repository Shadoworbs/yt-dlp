# yt-dlp CLI
A simple CLI tool to help you download yoututbe videos.

## Features
* Option to downlload videos in many formats eg. 144p, 480p, 720p, 1080p etc.
* Option to extract audio from a youtube video link.
* Supports playlist download.
* High quality audio.
